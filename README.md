# Book-shop Flutter App
- This app shows the collection of books in different categories.
- Users can add bookmark to books to make their own list.
- Add ,edit and delete review-rating on specific book and can also see others review on books.
- A particular book can be searched using book name or author name.
- Shimmer effect and Splash screen.
- Authentication is required.
